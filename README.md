### An open source files hash calculator for macOS.
 * MD5, SHA1, SHA256 and SHA512.
 * Support drag & drop.
 * Integrated with Finder context menu.
 * Support macOS 14 and later, with universal binary.
 * Support Dark Mode.
 * English and Simplified Chinese UI.

The icon "希" is from hash in Chinese "哈希".   

Project site: https://github.com/sunjw/fhash
